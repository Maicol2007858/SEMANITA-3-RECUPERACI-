# Gestión de Mascotas - Semana 3 POO

## Nombre
Michael Castillo

## Descripción
Este proyecto implementa dos soluciones para la gestión de mascotas utilizando Python:
1. Programación Tradicional
2. Programación Orientada a Objetos (POO)

Ambos programas manejan nombre, especie y edad de una mascota.

---

## Estructura del proyecto

programacion_tradicional/
- tradicional.py

programacion_poo/
- mascota.py
- main.py

---

## Diferencias entre los enfoques

### Programación Tradicional
- Usa funciones y variables
- No utiliza clases ni objetos
- Estructura más simple

### Programación Orientada a Objetos
- Usa clases y objetos
- Permite reutilización de código
- Mayor organización y escalabilidad

---

## Conclusión
La POO permite organizar mejor los programas mediante clases y objetos, mientras que la programación tradicional es más simple pero menos estructurada.
